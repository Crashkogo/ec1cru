Я работаю над веб-приложением на React + Node.js + PostgreSQL с Prisma. Это информационный сайт с админ-панелью для управления новостями, мероприятиями, акциями и готовыми решениями. Frontend: React 18, TypeScript, Vite, Tailwind CSS, Redux Toolkit, React Router, TinyMCE, Framer Motion, React Hook Form + Zod, reCAPTCHA v3. Backend: Express, Prisma, JWT, Nodemailer, Winston, express-fileupload. БД: PostgreSQL с моделями User, News, Events (с регистрациями), Promotions, ReadySolution + Program (many-to-many). Основной функционал: публичная часть с отображением контента и регистрацией на мероприятия (email-уведомления), админка с CRUD для контента, загрузка изображений с перемещением из temp. Особенности: SEO (metaTitle, metaDescription), slug с транслитерацией, роли (ADMIN, MODERATOR). Главная страница показывает последние 3 записи каждого типа. Код включает маршруты, контроллеры, компоненты (Home, NewsDetail, EventsDetail, PromotionsDetail, Dashboard, AdminMenu, News/NewsCreate/NewsEdit и др.). Текущая дата: 19 марта 2025.
